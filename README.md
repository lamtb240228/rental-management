# Rental Management System

Hệ thống quản lý phòng trọ giúp chủ trọ quản lý:

- Khu trọ và phòng
- Người thuê
- Hợp đồng thuê
- Chỉ số điện nước
- Hóa đơn
- Thanh toán
- Yêu cầu sửa chữa

## Technology Stack

### Backend

- Java 21
- Spring Boot 3.5.x
- Maven
- PostgreSQL 16
- Spring Data JPA
- Spring Security
- JWT
- Flyway

### Frontend

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- shadcn/ui
- TanStack Query
- Axios
- React Hook Form
- Zod
- React Router
